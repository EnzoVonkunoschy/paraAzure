Práctica de Circuito Elemental y Resistencia Eléctrica.

Lista de materiales para la práctica

  1. Fuente universal de alimentación (Generador Variable) con conectores.
  2. Plantilla de soldadura.
  3. Interruptor.
  4. Lámpara.
  5. Cable de conexiones.

Lista de herramientas e insumos que debe poseer el alumno para la práctica

  1. Soldador
  2. Tester
  3. Alicates
  4. Estaño

Preguntas que debe contestar antes de realizar la práctica

  • Materia, energía y cargas (26 de marzo).
  • Generadores (01 de abril).
  • Sentido de circulación de Corriente(8 de abril).
  • Unidad de Intensidad de Corriente Eléctrica (9 de abril).
  • Unidad de Intensidad de Corriente Eléctrica (10 de abril).

Condiciones para aprobar:
